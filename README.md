# Bird Tracker

A lightweight GPS + cellular wildlife tracker built around the STM32U031, SIM7080G (NB-IoT/LTE-M), and u-blox SAM-M10Q GNSS module.

## Repository structure

```
firmware/   STM32 firmware
hardware/   KiCad schematics and PCB layout
```
